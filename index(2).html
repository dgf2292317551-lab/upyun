服务器展示

本页面全面介绍服务器。

一、服务器概述

服务器（Servers）一般运行 Linux 或 Windows 服务器版本，用于支撑应用、网站、小程序等稳定运行。服务器特点：稳定、易扩展。

二、服务器分类与选型

  • 物理服务器：需自购设备、部署、机房带宽，适合测试环境及访问量大场景。
  • 公有云服务器：按需分配，无需管理物理设备，适合中小企业。
  • 私有云/虚拟化：自建环境，自定义高，数据安全性高。

常见品牌：

  • 物理服务器：Dell、华为、浪潮等
  • 云服务器：阿里云、腾讯云、华为云、AWS、GCP、Azure

三、目录结构与配置文件

  • /etc/hostname：主机名配置文件
  • /etc/hosts：主机名与 IP 映射
  • /etc/sysconfig/network-scripts/：网络配置文件
  • /etc/rc.local：开机自启动
  • /etc/fstab：磁盘挂载配置
  • /var/log/：日志目录（secure 登录日志、messages 系统日志）
  • /proc/：虚拟目录，存储内核及进程信息

四、主机名与网络配置

# 临时修改
hostnamectl set-hostname boot

# 查看主机名
hostname 或 cat /etc/hostname

# 永久修改
编辑 /etc/hostname 文件

# 网卡配置（固定 IP 示例）
BOOTPROTO=static
ONBOOT=yes
IPADDR=192.168.10.100
NETMASK=255.255.255.0
GATEWAY=192.168.10.2
DNS1=192.168.10.2

五、CPU 与内存信息

通过 lscpu 查看 CPU 架构、核心数、虚拟化支持。

通过 /proc/meminfo 查看内存信息，了解系统性能。

六、日志与排错

常用日志路径：

  • /var/log/secure：用户登录日志，分析暴力破解。
  • /var/log/messages：系统与服务异常日志。
